## テーマ別課題
極めて機密性が高い情報（電子カルテ情報など）をLLMがユーザとの対話内容に応じて参照し、必要があれば情報の更新を行うような対話アプリケーションを考えます。この際、機密情報をインターネット経由で外部に送信することが一切許されないとします。このようなケースでは、ローカルLLMとローカルデータベースを繋ぎ込むことが求められることがありますが、このときどのようなシステムアーキテクチャが考えられるか、図と文章で説明してください。また、可能であればその中で MCP (Model Context Protocol) を利用する場合についても検討し、その利点について解説してください。
