## テーマ別課題
単一入力を受け付け単一出力をする時系列システムに対する入出力データファイルを使って、当該システムの同定を行ってください。同定の過程は、別途指示される形式に沿ってレポートにまとめてください。

提供される10個の入出力データは、同一のシステムに対して、10パターンの異なる入力信号を入れた際の入力・出力が記録されています。これら入力・出力データの可視化・分析を通じて、このシステムの動特性を表現する適切な数理モデルを仮定、提供されたデータを活用してその数理モデルを推定してください。最後にテストデータとして与えられた入力信号に対する推定したモデルの応答を出力してください。


### データについて
データは`./data`フォルダに以下のように格納されています。

```
--- data
    ├── train_data
    |   ├── data_01.csv
    |   ├── data_02.csv
    |   ├── data_03.csv
    |     :
    |     :
    |   ∟ data_10.csv
     ∟ test_data
        ├── data_01.csv
        ├── data_02.csv
        ∟ data_03.csv
```
* フォルダ構成
    * `train_data` フォルダ: 学習用データ（10個のファイル）。
    * `test_data` フォルダ: テスト用データ（3個のファイル）。
* 各ファイルの形式はCSVで、各列は以下の通りです。
    * 1列目 `time`: 時刻。すべてのファイルでデータのサンプリングは等間隔。
    * 2列目 `input`: システムへの入力。
    * 3列目 `output`: システムの出力 。この出力はノイズを含んでいます。またテストデータについてはこの列はありません。
 
* 各ファイルは独立した試行（実験）で得られたデータですが、対象としているシステムは同一で、ノイズなども同じ条件下で発生しています。
* すべてのファイルにおいて、データ取得開始前のシステムは十分長い時間0が入力されていたものとしてください。
